# **App Name**: Vedang

## Core Features:

- Profile Showcase: Display a professional profile with key information and a personal photo.
- Publications List: Showcase recent academic papers with links to external resources.
- Essay Display: Present a collection of essays with engaging titles and brief summaries.

## Style Guidelines:

- Primary color: Midnight Blue (#2C3E50) for a professional and trustworthy feel.
- Background color: Light Gray (#F0F4F8) for a clean and modern look.
- Body font: 'Inter', a sans-serif font, will be used for body text, ensuring readability and a modern feel.
- Headline font: 'Lora', a serif font, for headlines, providing an elegant and intellectual touch.
- Use minimalist icons for social media links and navigation.
- Ensure a clean, responsive layout that is easy to navigate on all devices.
- Subtle transitions on hover for interactive elements to enhance user experience.
